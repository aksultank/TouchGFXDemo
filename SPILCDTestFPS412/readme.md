# What's this code
This code is sample of SPI LCD with Nucleo-F412ZG.

# Hardware

1. NUCLEO-F412ZG
2. HiLetGo 2.8 TFT LCD

# Connection

|Pin|Function|
|---|---|
|PA5|SCK|
|PA6|MISO|
|PA7|MOSI|
|PB15|RESET|
|PD14|CS|
|PC6|DC|
|PB8|LED|
